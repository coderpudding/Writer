# Writers I Like

![preview](preview.jpg)

## Description
A visual showcase of four writers who shaped the way I think —
Kafka, Nietzsche, Tolstoy, and Dostoevsky — each with a quote
that stuck with me.

## Built with
- HTML
- CSS
- JavaScript

## How to use
- Hover over any panel to reveal the quote
- Click on a panel to open a larger view
- Click anywhere outside to close
